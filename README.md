# 3-fase---1-Desafio-do-Challenge-Alura-Store

Alura Store - Análise de Desempenho das Lojas

Descrição do Projeto

Este projeto tem como objetivo ajudar o “Senhor João” a identificar qual loja da sua rede “Alura Store” apresenta menor eficiência, por meio da análise de dados de vendas, desempenho e avaliações dos clientes.

O projeto utiliza dados fictícios de quatro lojas e realiza análises como:

  ➢ Faturamento total por loja.
  
  ➢ Vendas por categoria de produto.
  
  ➢ Média de avaliação por loja.
  
  ➢ Produtos mais e menos vendidos.
  
  ➢ Frete médio por loja.
  
  ➢ Análise geográfica das vendas.

Além disso, são gerados gráficos e visualizações para facilitar a interpretação dos resultados


Estrutura do Projeto

O projeto contém os seguintes elementos:

  ➢ Dados: arquivos CSV contendo informações de vendas, produtos, avaliações, frete e localização das lojas.
  
  ➢ Códigos de análise: scripts Python utilizando pandas para manipulação de dados e “matplotlib”/”folium” para visualizações.
  
  ➢ Gráficos gerados:
  
     • Faturamento por loja (gráfico de pizza com porcentagens e valores em R$).
     
     • Média de avaliação por loja (gráfico de coluna + linha).
     
     • Frete médio por loja (gráfico de barra ou linha com valores em R$).
     
     • Distribuição geográfica das vendas (scatter plot ou mapa interativo com Folium).


Tecnologias e Bibliotecas Utilizadas

 ➢ Python 3.x
 
 ➢ Pandas → manipulação de dados CSV.
 
 ➢ Matplotlib → geração de gráficos (pizza, barra, linha).
 
 ➢ Folium → visualização geográfica interativa.
 
 ➢ Jupyter Notebook / Google Colab → execução do código e visualização das tabelas e gráficos.

 
Resultados e Insights

 ➢ Faturamento: Loja 1 apresentou o maior faturamento de R$ 1.534.509,12, enquanto Loja 4 apresentou o menor, com um valor de R$ 1.384.497,58.
 
 ➢ Vendas por categoria: Eletrônicos e móveis foram os mais vendidos em todas as lojas, com variações regionais.
 
 ➢ Avaliação dos clientes: A Loja 3 teve a melhor média de 4,00 de satisfação com, enquanto Loja 1 precisa melhorar, teve uma média de 3,98.
 
 ➢ Produtos mais e menos vendidos: Identificados os top 5 produtos mais vendidos e 5 menos vendidos em cada loja.
 
 ➢ Frete médio: Loja 1 apresentou maior custo médio de frete de R$ 34,69, indicando possível revisão logística.
 
 ➢ Distribuição geográfica: As vendas estão concentradas principalmente em específicas, destacando áreas de maior oportunidade ou necessidade de marketing local.
